1. Otvorte súbor `powmrESP32.ino` tu na GitHube a skopírujte jeho obsah.
2. Vložte kód do vášho **Arduino IDE**.
3. **Dôležité:** Na samom začiatku kódu (riadky 8 až 15) prepíšte vzorové údaje za vaše vlastné:
   * `WIFI_SSID` a `WIFI_PASS` (vaša Wi-Fi sieť)
   * `BLE_MAC` (Bluetooth MAC adresa vášho PowMr dongle)
   * `MQTT_PASS_STR` (heslo k vášmu MQTT brokeru)
4. Flashnite kód do mikrokontroléra ESP32.
